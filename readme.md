# 🌦 Weather & AQI Statistical Analysis - Gandhinagar

This Python project performs basic statistical analysis (Average and Median) on:

- 🌡 Temperature (°C)
- 💧 Humidity (%)
- 🌫 AQI (Air Quality Index)

for the last 10 days in Gandhinagar.

The program also stores the analysis results in a text file.

---

## 📌 Features

- Calculates **Average**
- Calculates **Median**
- Displays results in console
- Saves results to a text file automatically
- Simple and beginner-friendly code
- No external libraries required (uses built-in Python modules)
